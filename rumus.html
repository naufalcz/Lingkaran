    • Rumus Lingkaran

    Rumus lingkaran dalam website ini diantara lain ada : Persamaan lingkaran, garis singgung persekutuan luar, garis singgung persekutuan dalam, panjang busur, dan luas juring.

    Persamaan Lingkaran

    Titik Pusat (0,0)
    Rumus persamaan lingkaran jika memiliki titik pusat (0,0) yaitu : x² + y² = r²


    Titik Pusat (a,b)
    Rumus persamaan lingkaran jika memiliki titik pusat (a,b) yaitu : (x - a)² + (y - b)² = r².

    Garis Singgung Persekutuan Luar

    Rumus Garis Singgung Persekutuan Luar yaitu : d = √(s² - (R - r)²)



    Garis Singgung Persekutuan Dalam

    Rumus Garis Singgung Persekutuan Dalam yaitu : d = √(s² - (R + r)²)

    Panjang Busur

    Rumus Panjang Busur yaitu : L = (θ/360°) * 2πr

    Luas Juring

    Rumus Luas Juring yaitu : α/360° × πr²
